About
Journey Through Data is an interactive learning platform for non-experts about machine learning. This tutorial provides interaction when learning about machine learning algorithms and explains with real-life examples in order to enhance understanding.
Challenge
Compared to the rapid development of technology, there are not enough learning resources about technology such as machine learning or deep learaning that are easily accessible and understandable for non-experts. This results in using new services like chatbots or smart speakers, but not knowing how and what features are making them workable.

RESEARCH : INTERVIEW
I interviewed 5 non-experts who have at least tried to study machine learning and 1 machine learning engineer who works in a startup. I asked their experiences in studying machine learning or experiences in teaching machine learning to a non-expert.
The first 2 chapters that were designed were deleted in the end after going through several user tests. The chapters that dealt with supervised and unsupervised learning were reported that it was confusing to understand. Compared to the real life examples that comes after these chapters, it was hard to relate between the two sections. Instead of going through these chapters, I decided to only briefly mention about what data is and go straight into different algorithms.
I’ve taken various online & offline courses about machine learaning, but they all focus only on theories. I do not know how to apply them in real life.

Journey Through Data illustrates how I myself gone through the process of learning machine learning and applying the concept regarding how it can be utilized in real life services. As a designer who already studied about machine learning, I felt there was a massive shortage in this field, which led to designing an interactive learning platform.

This project gone through numerous iteractive prototypes, especially in creating the table of contents and the context how each chapter is developed. Each time it was tested to a non-expert, which relatively resulted in receiving good feedbacks from a point of view of a non-expert. It was also a project that challenged myself to learn CoffeeScript in Framer, and to design each interaction methods that will be intuitive for users.
